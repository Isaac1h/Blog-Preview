# Frontend Mentor - Blog preview card

J’ai réalisé une carte de prévisualisation de blog en HTML et CSS dans le cadre d’un défi Frontend Mentor. Je me suis inspiré du design fourni, disponible en versions mobile et desktop, et j’ai utilisé tous les fichiers nécessaires comme les images, les polices et le guide de styles. J’ai pris le temps de structurer mon HTML et de styliser chaque section du projet à ma façon pour rester fidèle au design tout en appliquant mes choix personnels.
